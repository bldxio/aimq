## ðŸ’¡ CONTEXT

**Good topics to research:**
- New libraries or frameworks being considered
- Best practices for technologies in use
- Design patterns for current problems
- Performance optimization techniques
- Security best practices
- Testing strategies

**Evaluation criteria:**
- Is it relevant to the project?
- Is it from a reputable source?
- Is it current and up-to-date?
- Can we apply it practically?
- Does it align with project values?

**Content structure:**
- Clear explanation of concept
- Why it matters for this project
- How to apply it (with examples)
- When to use / when not to use
- Links to sources and further reading

**Garden categories:**
- `.claude/patterns/` - New code patterns discovered
- `.claude/standards/` - Best practices to adopt
- `.claude/architecture/` - Design approaches to consider
- `.claude/quick-references/` - How-to guides for new tools
